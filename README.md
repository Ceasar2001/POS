## 🖥️ Features

- 📋 Product & Category Management
- 🛒 Order and Cart System
- 👨‍🍳 Staff & Role Permissions
- 📈 Sales Dashboard & Analytics
- 💾 Offline-first design (Coming Soon)
- 🔐 Secure login and JWT auth

- Node.js v20+

```bash
